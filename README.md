# vue-weather-app

## *[Watch Demo](https://kaiseridze.github.io/vue-test-task/ "Link to demo")*

###  Functionality:
* Fetch geolocation
* Add city
* Delete city
* Saving the state of dragging cities
* Saving the last added city as default
* Click on city in recents to fetch
### Stack:
1. Vue
2. VueX  
3. TypeScript
4. Vue-draggable
5. Scss
